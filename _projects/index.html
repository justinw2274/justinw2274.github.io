IS 445 Homework 5 Jekyll Webpage - UFO Sighting Visualizations

This project presents two interactive visualizations of UFO sightings using Python, Altair, and Vega-Lite. The first visualization is a scatter plot that maps the geographic distribution of UFO sightings, while the second visualization is a bar chart that shows the frequency distribution of reported UFO shapes.

Visualization 1: Scatter Plot of UFO Sightings

This scatter plot visualizes UFO sightings by plotting their latitude and longitude coordinates on a map. The datetime variable is encoded as a temporal field using the inferno sequential color schemeâ€”darker hues indicate more recent sightings, while lighter hues denote older events. Interactive tooltips display details such as the sighting date and coordinates, enhancing the spatial exploration of the dataset. In the Python notebook, I transformed raw date strings into datetime objects and filtered out records with missing or invalid data to maintain data quality.

Visualization 2: Bar Chart of UFO Shapes

This bar chart represents the frequency distribution of UFO shapes reported in the dataset. The x-axis encodes the UFO shape (a nominal variable) and the y-axis quantifies the number of sightings (a quantitative variable). I implemented an interactive hover effect that highlights the selected bar in steelblue while dimming the others, thereby assisting users in focusing on specific categories and comparing values effectively. Data transformations in Python included aggregating and grouping the UFO sighting data by shape.

Interactivity

For the bar chart, the interactive hover effect allows users to instantly highlight a specific UFO shape, which improves readability and comparisons between categories. This dynamic feedback elevates the interactivity beyond standard panning/zooming, making the visualization more engaging.
